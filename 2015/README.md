# Code For Denver - 2015 End-of-year-report

## Development:

- install boot from http://boot-clj.com/
- cd into 2015/ directory
- run $ boot dev
- visit localhost:3000

## Browser Connected REPL:

- boot dev
- boot repl --client
- (start-repl)

## Building Static Assets

- boot static

## Testing

- boot test
